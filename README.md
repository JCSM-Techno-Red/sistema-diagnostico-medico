# Sistema de Diagnóstico Médico

Sistema completo para cadastro de pacientes e diagnóstico médico com interface gráfica.

## Funcionalidades
- Cadastro completo de pacientes
- Sistema de diagnóstico com múltiplas doenças
- Histórico de diagnósticos
- Exportação de relatórios
- Backups automáticos

## Tecnologias
- Python 3.x
- Tkinter (interface gráfica)
- JSON para armazenamento de dados

## Como Executar
1. Instale Python 3.x
2. Execute: `python main.py`
3. Certifique-se que o arquivo `sintomas.json` está na mesma pasta

## Estrutura do Projeto
- `main.py` - Ponto de entrada
- `interface.py` - Interface principal
- `database.py` - Gerenciamento de dados
- `models.py` - Modelos de dados
- `engine.py` - Motor de diagnóstico
- `sintomas.json` - Banco de dados de sintomas e doenças
